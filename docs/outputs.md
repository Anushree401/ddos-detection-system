# Output, Graphs, and Evaluation

The system evaluates the performance of the decision engine visually using diagnostic plots located in `src/visualization/`. The combined output is typically saved as `diagnostics.png`.

## 1. Score Distribution (KDE Plot)

Generated by `distributions.py`, this graph visualizes the engine's ability to separate legitimate traffic from malicious traffic.

*   **Type**: Kernel Density Estimation (KDE) plot.
*   **X-Axis**: The calculated `heuristic_score` (0-100).
*   **Y-Axis**: Traffic Density.
*   **Colors**: Green represents ground-truth normal traffic; Red represents ground-truth attack traffic.
*   **Threshold Lines**:
    *   **Orange Dashed Line (Score 40)**: The "Suspicious" threshold. Traffic crossing this line warrants investigation.
    *   **Dark Red Dashed Line (Score 70)**: The "Attack" threshold. Traffic crossing this line is flagged with high confidence as a DDoS attack.

*Ideal Output:* You want to see a massive green peak on the left (low scores) and a distinct red peak on the right (high scores), with minimal overlap in the middle.

## 2. Detection Confusion Matrix

Generated by `confusion.py`, this heat map assesses the absolute accuracy of the engine against the labeled ground truth.

*   **True Attack (True Positive)**: Traffic correctly flagged as an attack.
*   **True Normal (True Negative)**: Legitimate traffic correctly allowed through.
*   **False Positive**: Normal traffic falsely flagged as an attack (Impacts user experience).
*   **False Negative**: Attack traffic missed by the engine (Impacts server availability).

## 3. Performance Metrics

While the graphs provide a visual summary, `metrics.py` calculates hard statistical numbers to validate the model:
*   **Precision**: When the engine flags an attack, how often is it actually an attack?
*   **Recall (Sensitivity)**: Out of all the real attacks, how many did the engine successfully catch?
*   **F1-Score**: The harmonic mean of precision and recall, providing a single metric for overall engine balance.
